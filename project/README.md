# Projektteil

Im Projektteil arbeiten Sie in drei Teams unabhängig voneinander an einem Datensatz und entwickeln jeweils eine eigene Klassifikationspipeline. Am Ende werden die Modelle der Teams in einem Meta-Classifier kombiniert.

## Teams
- **Team A** — `team_a/`
- **Team B** — `team_b/`
- **Team C** — `team_c/`

## Meta-Classifier
Die Ergebnisse aller Teams fließen in einen gemeinsamen Meta-Classifier ein.
