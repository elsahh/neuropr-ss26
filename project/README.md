# Projektteil

Im Projektteil arbeiten drei Teams unabhängig voneinander an einem MNE-Datensatz und entwickeln jeweils eine eigene Klassifikationspipeline. Am Ende werden die Modelle der Teams in einem Meta-Klassifikator kombiniert.

## Teams
- **Team A** — `team_a/`
- **Team B** — `team_b/`
- **Team C** — `team_c/`

## Meta-Klassifikator
Die Ergebnisse aller Teams fließen in einen gemeinsamen Meta-Klassifikator ein (`meta_classifier/`).
